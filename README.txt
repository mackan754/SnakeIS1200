
Snake Game till uno 32 chipkit

Kontroller:
--Välj ett spelläge med switcharna
--Höger: button 1
--Vänster: button 4
--Nedåt: button 2
--Uppåt: button 3
-- Om du förlorar starta ett nytt spel genom att klicka "Höger"

För att köra spelet:
--Öppna ett nytt msys2 fönster med mcb32 environment installerat
-- . /opt/mcb32tools/environment/
-- make
-- make install TTYDEV=/dev/ttyS4 (Siffran är beroende på din dators layout.)
